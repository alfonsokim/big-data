Introduccin

A continuacin se realizar el anlisis de los alumnos del Posgrado de Economa de la UNAM, Sede Facultad de Economa.

La informacin comprende el periodo 2006-2015, durante estos aos el Posgrado ha pasado por ms de 5 administraciones y por tanto la forma en la cual se ha llevado el control sobre la informacin de los alumnos es totalmente variada.

En la Reunin de Comit Acadmico celebrada en el mes de Febrero se le solicit al Posgrado que entregara la siguiente informacin para los alumnos de maestra y doctorado: * Nmero de alumnos por generacin * Nmero de alumnos por plan de estudios * Alumnos que han ingresado a ms de un plan de estudios * Nmero de alumnos titulados por ao * Mes en el que ms se titulan

Limpieza de los datos

Debido a no se cuenta con una metodologa para la captura de datos y a que la informacin se encontraba en varios archivos y formatos la nica forma de ligar los archivos fue mediante el nmero de cuenta del alumno.

Creacin de tablas

CREATE TABLE alumnos(
    cuenta TINYINT,
    apellido_paterno VARCHAR (12),
    apellido_materno VARCHAR (12),
    nombre VARCHAR (20),    
);

CREATE TABLE posgrado(
    cuenta TINYINT,
    anio TINYINT,
    semestre TINYINT,
    programa TINYINT,
    plan TINYINT,
    ingreso_plan TINYIN,
    tutor VARCHAR (40),
    egreso TINYIN,
    fecha_grado DATE
);
CREATE TABLE programa(
    programa TINYINT,
    nom_prog VARCHAR (25),
);
CREATE TABLE plan_est(
    plan TINYINT,
    nom_plan VARCHAR (25),
);
